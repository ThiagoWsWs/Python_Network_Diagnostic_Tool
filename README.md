# Python Network Diagnostic Tool

A simple Python-based network troubleshooting tool designed for IT support and network diagnostics.

## Features

* Ping test
* DNS lookup
* Port checking
* Connectivity diagnostics
* Log generation

## Technologies

* Python
* subprocess
* socket
* os

## Usage

```bash
python main.py
```
